HARI 1

Pengenalan HTML

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup untuk membuat halaman web. HTML bukan bahasa pemrograman, tapi bahasa markup yang memberitahu browser bagaimana menampilkan konten.

Kenapa Belajar HTML?

  • Dasar dari semua website di internet
  • Mudah dipelajari untuk pemula
  • Fondasi untuk belajar CSS dan JavaScript
  • Skill penting untuk web developer
HARI 1

Struktur Dasar HTML

Setiap dokumen HTML memiliki struktur dasar seperti ini:

<!DOCTYPE html> <html> <head> <title>Judul Halaman</title> </head> <body> <h1>Hello World!</h1> <p>Ini paragraf pertama saya.</p> </body> </html>

Penjelasan:

  • <!DOCTYPE html> - Deklarasi tipe dokumen
  • <html> - Root element
  • <head> - Informasi meta tentang dokumen
  • <body> - Konten yang terlihat di browser
HARI 1

Heading dan Paragraph

Heading (h1 - h6)

Heading digunakan untuk judul dan sub-judul. Ada 6 level heading:

<h1>Heading 1 - Paling Besar</h1> <h2>Heading 2</h2> <h3>Heading 3</h3> <h4>Heading 4</h4> <h5>Heading 5</h5> <h6>Heading 6 - Paling Kecil</h6>

Paragraph

<p>Ini adalah paragraf.</p> <p>Ini paragraf kedua.</p>

HARI 2

Link dan Gambar

Link (Anchor Tag)

<a href="https://google.com">Klik di sini</a> <a href="halaman2.html">Ke Halaman 2</a> <a href="mailto:email@example.com">Email Saya</a>

Gambar

<img src="foto.jpg" alt="Deskripsi gambar"> <img src="https://example.com/logo.png" alt="Logo" width="200">

Tips: Atribut alt penting untuk accessibility!

HARI 2

Lists dalam HTML

Unordered List (Bullet Points)

<ul> <li>Item pertama</li> <li>Item kedua</li> <li>Item ketiga</li> </ul>

Ordered List (Numbering)

<ol> <li>Langkah 1</li> <li>Langkah 2</li> <li>Langkah 3</li> </ol>

Semantic HTML

Semantic tags memberikan makna pada struktur halaman:

<header> <nav>Menu navigasi</nav> </header> <main> <article> <h1>Judul Artikel</h1> <p>Konten artikel...</p> </article> <aside>Sidebar</aside> </main> <footer> <p>Copyright 2025</p> </footer>
